aaa-btn

Clay CSS Button Recipes

Clay CSS follows Bootstrap 4’s component base + modifier pattern, similar to OOCSS. The base class, .btn, should only contain styles common to all .btn elements, such as border-style, border-width, display, font-size, and any other button related css resets.

The modifier, .btn-primary, provides the styles for the main variant of .btn, such as background-color, border-color, and color. The modifier, .btn-secondary, provides styles for the next .btn variant.

For Styled Theme

In this example, we will globally modify .btn-primary to match the purple button shown in Material Design. The first step is to create the file /src/css/_clay_variables.scss in your theme. Your Clay CSS variable overwrites will live there.

On the site, we can inspect the button’s styles. The styles are listed below:

/* Commented styles are already covered by Bootstrap's .btn class */
.btn {
    background-color: #26a69a;
    border-radius: 2px;
    /* border: none; */
    box-shadow: 0 2px 2px 0 rgba(0,0,0,0.14), 0 3px 1px -2px rgba(0,0,0,0.12), 0 1px 5px 0 rgba(0,0,0,0.2);
    color: #fff;
    /* cursor: pointer; */
    /* display: inline-block; */
    font-size: 14px;
    height: 36px;
    letter-spacing: .5px;
    line-height: 36px;
    /* outline: 0; */
    padding: 0 16px;
    /* -webkit-tap-highlight-color: transparent; */
    /* text-align: center; */
    /* text-decoration: none; */
    text-transform: uppercase;
    -webkit-transition: background-color .2s ease-out;
    transition: background-color .2s ease-out;
    /* vertical-align: middle; */
}

.btn:hover {
    background-color: #2bbbad;
    box-shadow: 0 3px 3px 0 rgba(0,0,0,0.14), 0 1px 7px 0 rgba(0,0,0,0.12), 0 3px 1px -1px rgba(0,0,0,0.2);
}

.btn:focus {
    background-color: #1d7d74;
}

.btn:disabled {
    background-color: #DFDFDF !important;
    box-shadow: none;
    color: #9F9F9F !important;
    /* cursor: default; */
    /* pointer-events: none; */
}

We now have the styles we need.

$enable-shadows: true;

// The base button variables
$btn-border-radius: 0.125rem;
$btn-border-width: 0;
$btn-box-shadow: 0 2px 2px 0 rgba(0,0,0,0.14), 0 3px 1px -2px rgba(0,0,0,0.12), 0 1px 5px 0 rgba(0,0,0,0.2);
$btn-font-size: 0.875rem;
$btn-line-height: 2.25rem;
$btn-padding-x: 1rem;
$btn-padding-y: 0;


$btn-secondary: (
    bg: #26a69a,
    hover-bg: #2bbbad,
    focus-bg: #1d7d74,
    active-bg: #1d7d74,
    active-box-shadow: $btn-box-shadow, // This is broken as of 2.20.x
    disabled-bg: #dfdfdf,
    disabled-color: #9f9f9f,
);
