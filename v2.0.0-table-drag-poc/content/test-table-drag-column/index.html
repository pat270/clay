Start Drag

The table divider isn't supported for now.
  1. Remove the class show-quick-actions-on-hover on table.
  2. Copy the each th and td in the Format Column and put it inside a table template.
  3. Get the width of the column (84px in desktop) and set it on the table element. Get the height of each row and set it on the respective tr inside the table template.
  4. Calculate the position of the column and set .table-column-drag-container left css property to match.
  5. Add class table-column-dragging to each table cell in the column.
