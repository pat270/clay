v2.3.4 October 3, 2018

  • No Clay CSS changes

v2.3.3 September 24, 2018

  • Fixes #1105, #1147 - ClayCSS: Subnav Tbar Primary should support labels, break to new line, have disabled state, and .component-link color should have more contrast
  • ClayCSS: #1105 Mixin clay-label-size added options to configure margin-bottom, margin-left, margin-right, margin-top
  • ClayCSS: #1105 Mixin clay-tbar-variant added options to configure $padding-x, $padding-y, $item-justify-content, $btn-monospaced-margin-x, $btn-monospaced-margin-y, $link-monospaced-margin-x, $link-monospaced-margin-y, $label-size
  • ClayCSS #1105 Tbar added option to break to new lines with .tbar-nav-shrink, .tbar-nav-wrap, .subnav-tbar-inline-xs-down, .subnav-tbar-inline-sm-down, .subnav-tbar-inline-md-down, .subnav-tbar-inline-lg-down, .subnav-tbar-inline-xl-down
  • ClayCSS: #1105 Label disabled .close hover state should look the same
  • ClayCSS: #1105 Mixin clay-link add option to configure disabled-text-decoration
  • ClayCSS: #1105 New mixin clay-label-variant for creating custom label colors
  • ClayCSS: #1147 Subnav Tbar Primary changed .component-link color to have more contrast and added .subnav-tbar-primary.subnav-tbar-disabled colors
  • ClayCSS: #1105 Tbar renamed .subnav-tbar-inline-*-down to .tbar-inline-*-down
  • Fixes #1105 - ClayCSS: Mixin clay-text-typography and option to pass in display, max-width, and word-wrap properties

v2.3.2 September 3, 2018

  • No Clay CSS Changes

v2.3.1 September 3, 2018

  • ClayCSS: Fixes #1125 Atlas Forms changing individual border-widths for an input doesn’t apply
  • ClayCSS: #1125 Forms added option to configure $input-border-style and declare border-style, border-color separately so we can size borders individually
  • ClayCSS: Fixes #1123 Use map-merge pattern for any Sass map variable with default values

v2.3.0 August 28, 2018

  • ClayCSS: Fixes #1090 Modal Variant Mixin $footer-border-color declared twice
  • Update: Fixes #1088 List Group variables $list-group-text-link and $list-group-subtext-link should follow same pattern as $list-group-title-link
  • ClayCSS: Fixes #1085 Use :hover :focus pseudo classes instead of Bootstraps mixin
  • New: Fixes #1083 Custom Form added $custom-control-indicator-checked-active-bg, $custom-control-indicator-checked-active-border-color, $custom-control-indicator-checked-disabled-bg, $custom-control-indicator-checked-disabled-border-color, $custom-radio-indicator-checked-disabled-border-color
  • Update: #1083 Custom Form deprecate old variable names $custom-control-indicator-active-checked-bg, $custom-control-indicator-active-checked-border-color, $custom-control-indicator-disabled-checked-bg, $custom-control-indicator-disabled-checked-border-color, $custom-radio-indicator-disabled-checked-border-color and map them to the new ones
  • New Fixes #1065 Table added helpers .table-cell-minw-50, .table-cell-minw-75, .table-cell-minw-100, .table-cell-minw-150, .table-cell-minw-200, .table-cell-minw-250, .table-cell-minw-300, .table-cell-minw-350, .table-cell-minw-400, .table-cell-ws-normal, and .table-cell-ws-nowrap for customizing table columns
  • New: #1065 Table added option to configure $table-valign-top-body-cell-padding-top, $table-valign-bottom-body-cell-padding-bottom and align .quick-action-menu to the top or bottom
  • Update: Fixes #1053 Atlas Card box-shadow should match Lexicon card box shadow better
