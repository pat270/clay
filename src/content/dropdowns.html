    Dropdown wide / full

    Use .dropdown-wide with .dropdown to make the dropdown menu big. The default width is 500px. Use .dropdown-full to create a dropdown menu as wide as its relative parent. See dropdown examples.

    Dropdown Alignment

    The dropdown menu is, by default, aligned to the left side of the parent container. It can be aligned to the right or center by adding the classes .dropdown-menu-right or .dropdown-menu-center, respectively.

    You can center the dropdown menu in browsers that don't support css transforms by setting a negative margin-left equal to the width of the dropdown-menu divided by 2. For left-side and right-side dropdown-menus vertically align them by setting a negative margin-top equal to the height of the dropdown-menu divided by 2.

    The dropdown menu can also be aligned so the caret is aligned to center on the parent container. This is done by adding the .caret-aligned to the parent container. This can also be used with the .dropup class. This cannot be used with .dropdown-menu-center as the caret is already centered on the parent container.

    Note that vertical alignment (use with .dropdown-menu-side) only works with equilateral container elements as CSS calculates margin-top and margin-bottom values relative to the parent width, not height.

    Removing the Dropdown Arrow

    Add .dropdown-menu-no-arrow to .dropdown-menu or to a parent container to remove arrow styles from a dropdown-menu.
