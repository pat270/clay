Migrating to Lexicon v2

Migrating to Bootstrap v4

Global

  • Bootstrap 4 converted from using px to rem as their primary CSS unit. We removed sizing for small screens and large screens in Atlas accordingly. You can adjust the font-size on the html element to adjust the size of Lexicon components.
  • Atlas removed $font-size-desktop-base, $font-size-desktop-large, $font-size-desktop-small, $font-size-desktop-h1, $font-size-desktop-h2, $font-size-desktop-h3, $font-size-desktop-h4, $font-size-desktop-h5, $font-size-desktop-h6.

Mixins

  • Bootstrap 4 removed the following mixins: animation, animation-delay, animation-direction, animation-duration, animation-fill-mode, animation-iteration-count, animation-name, animation-timing-function, backface-visibility, box-sizing, content-columns, hyphens, opacity, perspective, perspective-origin, rotate, rotateX, rotateY, scale, scaleX, scaleY, skew, transform-origin, transition-delay, transition-duration, transition-property, transition-timing-function, transition-transform, translate, translate3d, user-select

Grid System

  • Bootstrap 4 converted media query breakpoints to $grid-breakpoints sass map. You must use Bootstrap 4's breakpoints mixins or create media queries using map-get($grid-breakpoints, md).
  • Bootstrap 4 removed variables: $screen-xs, $screen-xs-min, $screen-phone, $screen-sm, $screen-sm-min, $screen-tablet, $screen-md, $screen-md-min, $screen-desktop, $screen-lg, $screen-lg-min, $screen-lg-desktop, $screen-xs-max, $screen-sm-max, $screen-md-max, $grid-float-breakpoint, and $grid-float-breakpoint-max

Buttons

  • Bootstrap 4 renamed .btn-default to .btn-secondary
  • Bootstrap 4 removed .btn-xs
  • Bootstrap 4 removed $line-height argument from button-size mixin.
  • Bootstrap 4 moved button padding to variables $btn-padding-x and $btn-padding-y.
  • Bootstrap 4 moved button border-radius to variables $btn-border-radius, $btn-border-radius-lg, $btn-border-radius-sm.
  • Atlas removed .btn-xs
  • Atlas removed variables: $btn-line-height, $btn-desktop-xs-font-size, $btn-desktop-xs-padding-vertical, $btn-desktop-xs-padding-horizontal, $btn-desktop-sm-font-size, $btn-desktop-sm-padding-vertical, $btn-desktop-sm-padding-horizontal, $btn-desktop-lg-font-size, $btn-desktop-lg-padding-vertical, $btn-desktop-lg-padding-horizontal, $btn-desktop-font-size, $btn-desktop-padding-vertical, $btn-desktop-padding-horizontal, $btn-xs-font-size, $btn-xs-padding-horizontal, $btn-xs-padding-vertical, $btn-xs-border-radius
  • Atlas renamed $btn-padding-horizontal, $btn-padding-vertical, $btn-lg-padding-horizontal, $btn-lg-padding-vertical, $btn-sm-padding-horizontal, $btn-sm-padding-vertical, $btn-lg-border-radius, $btn-sm-border-radius.
