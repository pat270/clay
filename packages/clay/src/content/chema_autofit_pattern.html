Autofit Pattern

Liferay has finally dropped support for IE9 which gives us full access to flexbox. One of the more interesting patterns we have developed is the Autofit Pattern. It is now easier to make content expand to fill remaining space and vertically align content without having to resort to floats, clunky grid columns, or margin/padding hacks.

The classes you need to use are `.autofit-row`, `.autofit-col`, and `.autofit-col .autofit-col-expand`. In the example below we use this pattern with the help of `.component-title` and `.btn .btn-sm`. The font size and weight on `.component-title` can be changed and alignment with the pencil icon will remain the same.

Autofit Helpers

Use the class combo `autofit-float autofit-padded autofit-row` on the parent element to mimic floating content when the screen size is too small and to add some space between items.

One catch to `.autofit-col` is the use of flex-direction: column which turn all direct descendants into block level elements. If you want to display items inline, such as labels or buttons, you will need to wrap them in a `div`. We created a special class `.autofit-section` for items inside `.autofit-col`.
